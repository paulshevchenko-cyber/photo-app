# 📷 Фото на документы РФ

Веб-приложение для съёмки фотографий на российские документы прямо с телефона.
Работает в браузере без установки — один HTML-файл.

🔗 Живая версия: https://paulshevchenko-cyber.github.io/photo-app/

## Поддерживаемые документы

| Тип | Размер | Размер в пикселях (300 DPI) |
|-----|--------|-----------------------------|
| Паспорт РФ / Загранпаспорт / ВНЖ | 35×45 мм | 413×531 |
| Виза (Шенген, US, UK, Китай) | 35×45 мм | 413×531 |
| Военный билет / Приписное | 30×40 мм | 354×472 |

## Функции

- Прицел с овалом лица по **ГОСТ Р 70997-2023** (лицо 70–80% высоты кадра)
- Направляющие: темя, линия глаз, подбородок
- Авто-кроп в правильные пропорции
- **Выход 35×45 мм @ 300 DPI** с DPI-метаданными в JFIF — фото правильно печатается на любом принтере или фотокиоске
- **Печатный лист** 10×15 см или A4 с метками реза (через системный диалог печати → Save as PDF)
- **Загрузка из галереи** — обработать готовое фото без камеры
- **Портретный режим телефона** — если камера поддерживает (Chrome `backgroundBlur` constraint)
- **Удаление фона ИИ** опционально (ImgLy CDN)
- **👶 Детский режим** — серия 5 фото или загрузка 4–12 фото из галереи объединяются AI (Google Nano Banana) в одно фронтальное фото с **верификацией пропорций лица ±5% и возраста ±2 года**
- Логи и бэкапы последних 5 фото в IndexedDB
- Места под рекламу зарезервированы (включаются `?ads=1`)

## Использование

### Вариант 1 — GitHub Pages

1. Создайте публичный репозиторий
2. Положите `index.html` в корень
3. Settings → Pages → Source → ветка `main`, папка `/ (root)`
4. Откройте `https://<user>.github.io/<repo>/`

### Вариант 2 — локально

Камера требует HTTPS или `localhost`. Откройте по `file://` приложение покажет понятный экран с инструкцией.

```
python3 -m http.server 8000
# затем открыть http://localhost:8000/
```

## Детский режим (AI)

Маленький ребёнок не позирует — поэтому есть два способа:

1. **📸 Серия 5 фото** — приложение делает 5 снимков за ~3 секунды, отбракует кадры без лица, отправляет остаток в Gemini 2.5 Flash Image (Nano Banana) который собирает из них одно фронтальное фото.
2. **📁 Загрузка 4–12 фото** — любые фото ребёнка с разных ракурсов (день рождения, прогулка). Рекомендуем 8 фото.

После генерации AI результат **проверяется**:
- Межзрачковое расстояние, eye-to-chin, nose-to-mouth не должны отличаться от исходных более чем на 5% (серия) / 8% (загрузка) — иначе fallback на лучший кадр.
- Возраст на AI-результате не должен отличаться от исходного более чем на 2 года.

### API-ключ Gemini

Нужен ключ от Google AI Studio: https://aistudio.google.com/apikey

Ключ хранится **только в этом браузере** (`localStorage`). Не публикуйте копию приложения вместе со своим ключом — его украдут.

В будущем планируется альтернативный режим — AI через Telegram-бот (ключ на сервере). Архитектурный задел уже заложен (см. `TelegramProxyBackend` в коде).

## Печать

В превью готового фото выберите 10×15 или A4 → «⎙ Лист». Откроется системный диалог печати с раскладкой и метками реза. В iOS Safari и Android Chrome там есть «Сохранить как PDF».

## Проверка DPI

```
identify -verbose photo.jpg | grep -iE 'resolution|geometry'
# Resolution: 300x300, Units: PixelsPerInch, Geometry: 413x531

exiftool photo.jpg | grep -iE 'x resolution|y resolution|resolution unit'
# X Resolution: 300, Y Resolution: 300, Resolution Unit: inches
```

## Логи и бэкапы

В настройках (⚙) есть:
- **⤓ Экспорт лога** — скачать `.txt` с последними 200 событиями (для отладки)
- **🗂 Бэкапы** — последние 5 готовых фото из IndexedDB; кнопка «↺» возвращает фото в превью

## Архитектурные особенности

- **`Platform` адаптер** — изолирует web-специфику (`localStorage`, `navigator.share`, `window.open` для печати). При портировании в Android (Capacitor/WebView) заменяется только этот слой.
- **`AiBackend`** — две реализации:
  - `GeminiDirectBackend` — прямой вызов `generativelanguage.googleapis.com` из браузера. Текущий рабочий путь.
  - `TelegramProxyBackend` — заглушка для будущего: бот на сервере держит ключ, браузер шлёт фото боту.
- **`Log` ring buffer** в `localStorage` на 200 событий — диагностика без F12.
- **`Backup` в IndexedDB** — последние 5 финальных фото + последняя серия для повторных AI-попыток.

## Технологии

- Vanilla JS (один HTML-файл), Canvas 2D
- MediaPipe Face Detection (CDN, опционально — при недоступности приложение работает вручную)
- ImgLy background-removal 1.4.5 (CDN, опционально)
- Google Gemini 2.5 Flash Image (Nano Banana) для детского режима
