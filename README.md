# 📷 Фото на документы РФ

Веб-приложение для съёмки фотографий на российские документы прямо с телефона.  
Работает в браузере без установки — один HTML-файл, никаких зависимостей.

## Живой пример

👉 [Открыть приложение](https://your-username.github.io/photo-app/)

## Поддерживаемые документы

| Тип | Размер |
|-----|--------|
| Паспорт РФ / Загранпаспорт / ВНЖ | 35×45 мм |
| Виза (Шенген, US, UK, Китай) | 35×45 мм |
| Военный билет / Приписное | 30×40 мм |

## Функции

- Прицел с овалом лица согласно **ГОСТ Р 52425-2005** (лицо 70–80% высоты кадра)
- Направляющие линии: темя, линия глаз, подбородок
- Автоматическая обрезка в правильные пропорции
- Переключение камер (фронтальная ↔ основная)
- Сохранение фото на устройство (JPEG)
- Памятка по требованиям прямо в интерфейсе

## Использование

### Вариант 1 — GitHub Pages (рекомендуется)

1. Создайте публичный репозиторий
2. Скопируйте `index.html` в корень (или папку `docs/`)
3. В Settings → Pages → Source выберите ветку `main` / папку `/ (root)` или `/docs`
4. Открыть ссылку вида `https://username.github.io/repo-name/`

### Вариант 2 — локально

Просто откройте `index.html` в браузере Chrome/Firefox.  
⚠️ Камера требует HTTPS или `localhost` — локальный файл может не получить доступ к камере в некоторых браузерах.

## Технологии

- React 18 (CDN, без сборки)
- Babel Standalone (JSX в браузере)
- Web API: `getUserMedia`, `Canvas`, `ResizeObserver`

## Требования к фото (ГОСТ Р 52425-2005)

- Размер: 35×45 мм (паспорт/виза), 30×40 мм (военный билет)
- Лицо занимает 70–80% высоты снимка
- Фон белый или светло-серый, однотонный
- Взгляд прямо в объектив, нейтральное выражение
- Без теней, чёткое изображение
