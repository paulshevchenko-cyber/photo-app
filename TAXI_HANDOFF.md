## Что уже сделано

Создан файл `taxi.html` — однофайловая HTML-игра «Симулятор такси» для ребёнка, без зависимостей, открывается прямо в браузере. Базируется на предыдущей игре-лабиринте (`car_maze_kids_…html`).

### Реализованные фичи

- **Карта 600×900 (портрет)**: 3 горизонтальные + 3 вертикальные дороги по 2 полосы каждая (правостороннее движение), кольцевая развязка в центре, дома между улицами с окнами.
- **Управление** (относительные стрелки, а не компас — для ребёнка понятнее):
  - ▲ — поехать (отпустить тормоз)
  - ▼ — тормоз (зажать = стоять)
  - ◀ ▶ — поворот налево / направо на ближайшем перекрёстке
  - Свайп по канвасу — резерв
  - Клавиатура: стрелки/WASD, пробел — пауза, N — навигатор
- **Знаки**: STOP, «уступи», кольцо, пешеходный, главная, ограничение скорости. При первом приближении игра ставится на паузу, показывается крупный знак и объяснение. Знакомые знаки выводятся только тостом.
- **Светофоры** на двух перекрёстках (NW и SE). AI тормозит на красный.
- **AI-машины**: держат дистанцию от впереди идущих (включая игрока — не врежутся сзади), не въезжают на встречку, на кольце ездят против часовой стрелки (CCW).
- **Пассажиры**: 6 точек посадки/высадки на перекрёстках. Подбираешь 👤, везёшь к 🏁, +20 ⭐.
- **Очки**:
  - −15 лобовое (выезд на встречку при обгоне), −10 фронтальная авария, −5 притирка
  - −8 проезд на красный, −3 езда по встречке
  - +20 за доставку
  - +5 каждые 30 сек без нарушений (за 5 минут чистой езды ~+50)
- **Уровни**: 1 → нужно 30 ⭐, дальше +25 за уровень и больше AI-машин.
- **Голосовой навигатор** через Web Speech API (русский язык, переключается кнопкой 🔊). Подсказывает повороты типа «На следующем перекрёстке направо». Состояние сохраняется в localStorage. Если фраза длинная (>35 символов) — игра автоматически паузится, пока озвучивается.
- **Ручная пауза** (⏸): останавливает игру, оверлей «Продолжить».
- localStorage сохраняет: рекорд лучших, статус навигатора, увиденные знаки (чтобы не объяснять одно и то же 100 раз).

### Учтённые уроки из прошлого handoff (лабиринт)

D-pad на Android в лабиринте не работал. В такси-симе:
- **Pointer Events** (`pointerdown` + `pointerup` + `pointercancel` + `pointerleave`) с fallback на `touchstart`/`mousedown`. Универсально для touch и mouse.
- Кнопки 56×56px, в отдельной flex-зоне под канвасом (не поверх).
- `100dvh` + `env(safe-area-inset-bottom)`.
- `touch-action: manipulation` на кнопках.
- Свайп по канвасу как полноценная альтернатива.

## Что НЕ сделано / можно улучшить

- Нет нормального path-планирования для навигатора — он подсказывает только ближайший поворот «к цели», без полного маршрута через граф. Иногда подсказка может вести в тупик через одностороннюю улицу.
- AI на нерегулируемых перекрёстках не уступают друг другу по правилу «помеха справа» — могут крашиться между собой (на игроке это не сказывается, но визуально иногда странно).
- Кольцо нарисовано визуально, но физически это всё ещё 2×2 кластер перекрёсточных клеток. Игроку сложно «проехать по кругу» — поворот за поворот. Можно упростить или сделать настоящее непрерывное круговое движение.
- Звуки минимальные (бипы через WebAudio). Можно добавить полноценный фоновый шум двигателя.
- Спрайты машин ~16×22px (на 30px клетках), для совсем маленького ребёнка можно увеличить до 20×26.
- Знаки на канвасе мелкие (14px), их сложно заметить в потоке — можно увеличить.
- Нет анимации поворота — машина мгновенно меняет угол на 90° (хоть и сглажено через `visualAng`).

## Известные баги / шероховатости

- При повороте в первой ячейке перекрёстка машина может остаться на встречке, поэтому повороты разрешены только когда выходная клетка лежит в правильной полосе (см. `legalDirs` + `turnOk` в `updateCar`). Если игрок жмёт поворот слишком поздно, машина проедет перекрёсток прямо.
- `nextDir` сбрасывается через 4 секунды, чтобы не висел.
- speechSynthesis на iOS Safari иногда не запускается без явного user gesture — первая фраза может «съесться». На Android Chrome работает стабильно.

## Промпт для новой сессии Claude Code в Games-for-sons

После того как `taxi.html` и `TAXI_HANDOFF.md` лежат в новом репо, запустите Claude Code в `Games-for-sons` и вставьте:

```
В этом репо taxi.html — однофайловая HTML-игра «Симулятор такси»
для моего сына. Сначала прочитай TAXI_HANDOFF.md — там полное
описание что сделано, что не сделано, и какие были баги в прошлой
версии.

Сейчас нужно: открой игру в браузере (или эмуляторе), пройди один
уровень и проверь что работает. Особое внимание:
- D-pad кнопки реагируют на тап на мобильном (в прошлой игре-лабиринте
  это была главная проблема)
- Поворот налево/направо действительно выводит в правильную полосу
- Знаки срабатывают для нужного направления
- Кнопка ⏸ паузы и тумблер 🔊 навигатора работают

Если что-то поломано — поправь минимально-инвазивно, не переделывая
всю архитектуру. Центральная функция движения — updateCar.
```
